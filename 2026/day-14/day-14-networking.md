# Networking 

## OSI model 
it is open system interconnection (OSI) it is used to data travel from one computer to another computer. thre are 7 layers such as

7. Application layer :- user interacts here
6. Presentation layer :- data formatting , encryption  , compress
5. Session layer :- managing connection
4. Transport layer :- End to End delivery
3. Network layer :- logical address
2. Data link :- MAC address , error detection
1. Physical layer :- send row data through cables/wifi


## TCP/IP model 
Its focus on real communication .its combine of the OSI model .it hs 4 layer that is
4.Application layer
3.transport layer
2.Internet
1. link
